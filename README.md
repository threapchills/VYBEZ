# VYBEZ

Emergent ancient ambient chill

A browser-based living instrument: seven spirit musicians in a painted valley at perpetual dusk, playing endless interlocking polyrhythms from one shared scale and one shared clock. You never perform; you steer, by touching things in the scene. Nothing is saved, nothing repeats, and the music exists only while the valley is open.

Live at [threapchills.github.io/VYBEZ](https://threapchills.github.io/VYBEZ/).

## Running it locally

```
npm install
npm run dev
```

`npm run test`, `npm run typecheck` and `npm run lint` keep the gates. The full brief lives in [HANDOFF.md](HANDOFF.md); build decisions are logged in [CLAUDE.md](CLAUDE.md).
